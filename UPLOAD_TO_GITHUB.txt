1) Giải nén ZIP.
2) Tạo repository GitHub Public.
3) Upload TOÀN BỘ nội dung bên trong thư mục; index.html phải ở thư mục gốc.
4) Settings > Pages > Deploy from a branch > main > /(root) > Save.
5) Chờ 1-5 phút, mở link GitHub Pages và kiểm tra bằng cửa sổ ẩn danh.

Hướng dẫn chi tiết nằm trong README.md.
